# Account Login (no cookie)
基本帳號密碼驗證功能
# 基本功能
+ 輸入正確帳號密碼組合即可登入
+ 輸入錯誤帳號密碼組合無法登入，提示錯誤訊息

# 安裝方式
1.打開 terminal 將專案 clone 到本地電腦
```
git clone https://github.com/LIN-CHANG-YI/Login.git
```
2.進入專案資料夾
```
cd Login
```
3.安裝npm
```
npm install
```
4.安裝nodemon
```
npm install nodemon
```
5.啟動程式
```
npm run start
或
npm run dev
```
6.成功執行
```
在 terminal 可以看到 Express is listening on localhost:3000
```
7.開啟瀏覽器
```
網址列輸入localhost:3000
```
